Dean's Adventure

Wanting to explore and understand more about UX design, I decided to create an interactive game where the user dictates a character's fate. In Dean's Adventure each choice the user makes is crucial to the story where it could either save or kill Dean. I wanted to design this game so that it would give off a retro vibe, I decided that the style and the art had to have an 8-bit look.
